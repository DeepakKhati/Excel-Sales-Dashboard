## Dashboard Preview
![Sales Dashboard](Sales_Dashboard_Preview.png)

## Project Overview
Built an interactive sales dashboard analyzing 400+ orders 
to uncover revenue patterns across regions, cities and 
product categories.

## Dataset
- Records: 400+ orders
- Period: 2025 (12 months)
- Tool Used: Microsoft Excel

## Features
- 8 dynamic slicers (Date, Region, City, Category, 
  Sub-Category, Payment Mode, Sales Channel, 
  Delivery Status)
- KPI cards for Total Sales and Total Profit
- 6 chart types for multi-dimensional analysis
- Fully interactive — all charts update with slicers

## Business Questions Answered
1. Which region generates the highest revenue?
2. Which product category has the most demand?
3. What is the monthly revenue trend?
4. Which city contributes most to total sales?
5. What is the split between payment modes?
6. What percentage of orders are delivered vs returned?
